    - 3️⃣ What is the difference between map(), filter(), and forEach()?
    - Ans : Here is my all differents :
    Use map() when you want to create a new array by applying a function to every element of the original array. It does not change the original array.
    Use filter() when you want to create a new array containing only the elements that pass a specific condition.
    Use forEach() when you want to do something with every element (like logging to the console or saving to a database), but you don't need a new array back.
